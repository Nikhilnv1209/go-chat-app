# Story: Real-Time Messaging (Chat)

**Feature ID**: F-FCH
**Epic**: Chat
**Status**: DONE
**Owner**: Frontend Agent

---

## User Stories
*   **F-FCH-01 (Must)**: As a user, I want messages to appear *instantly* (Optimistic UI) when I send them.
*   **F-FCH-02 (Could)**: As a user, I want to see **who is Online** via a green status dot.
*   **F-FCH-03 (Should)**: As a user, I want to **delete a direct message conversation** so I can clear my chat list.

---

## Acceptance Criteria
- [x] Messages list renders correctly.
- [x] Sending a message appends it immediately to the list (Optimistic UI).
- [x] Incoming messages appear in real-time via WebSocket.
- [x] Avatars display presence status (online/offline).
- [ ] Context menu/button to delete conversation.
