# Story: Read Receipts

**Feature ID**: F-FRR
**Epic**: Read Receipts
**Status**: TODO

## User Stories
*   **F-FRR-01 (Should)**: As a user, I want to see **checkmarks** indicating if my message is Sent, Delivered, or Read.

---

## Acceptance Criteria
- [ ] Messages show single grey tick (Sent).
- [ ] Messages show double grey tick (Delivered).
- [ ] Messages show double blue tick (Read).
- [ ] Ticks update in real-time.
