# Story: The Inbox (Dashboard)

**Feature ID**: F-FDB
**Epic**: Dashboard
**Status**: IN_PROGRESS
**Owner**: Frontend Agent

---

## User Stories
*   **F-FDB-01 (Must)**: As a user, I want a **sidebar list** of all my Conversations (DMs & Groups) sorted by recent activity.
*   **F-FDB-02 (Should)**: As a user, I want to see **"Unread" badges** so I know where to focus.
*   **F-FDB-03 (Could)**: As a user, I can toggle between **Light and Dark mode**.
*   **F-FDB-04 (Should)**: As a user, I want a modern, **floating sidebar layout** that optimizes space on larger screens.
*   **F-FDB-05 (Should)**: As a user, I want to **organize my chats into custom folders** (e.g., Work, Friends) to keep my workspace tidy.

---

## Acceptance Criteria
- [x] Sidebar displays list of conversations (DMs and Groups).
- [x] List is sorted by `last_message_at` (most recent top).
- [x] Clicking a conversation navigates to the chat view.
- [x] "Unread" badge appears when `unread_count > 0`.
- [x] Sidebar collapses/floats responsibly.
- [ ] Folder management UI is available.
