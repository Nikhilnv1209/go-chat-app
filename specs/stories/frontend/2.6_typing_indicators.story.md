# Story: Typing Indicators

**Feature ID**: F-FTI
**Epic**: Typing Indicators
**Status**: DONE
**Owner**: Frontend Agent

---

## User Stories
*   **F-FTI-01 (Should)**: As a user, I want to see a **"User is typing..." animation** when someone is typing to me.

---

## Acceptance Criteria
- [x] Typing triggers `typing_start` event.
- [x] Stopping typing (or blur) triggers `typing_stop` event.
- [x] UI displays typing indicator for the specific conversation.
- [x] Indicator disappears after timeout or stop event.
