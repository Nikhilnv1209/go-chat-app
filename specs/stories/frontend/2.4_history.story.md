# Story: History

**Feature ID**: F-FHS
**Epic**: History
**Status**: TODO
**Owner**: Frontend Agent

---

## User Stories
*   **F-FHS-01 (Should)**: As a user, I want to scroll up to see older message history (**Infinite Scroll**).

---

## Acceptance Criteria
- [ ] Initial load fetches first page of messages (e.g. 50).
- [ ] Scrolling to top triggers fetch of next page.
- [ ] Scroll position is maintained after loading older messages.
- [ ] Loading spinner appears during fetch.
