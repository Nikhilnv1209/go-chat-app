# Story: Authentication & Onboarding

**Feature ID**: F-FAU
**Epic**: Authentication
**Status**: DONE
**Owner**: Frontend Agent

---

## User Stories
*   **F-FAU-01 (Must)**: As a user, I want to **Sign Up** with email/password so I can create an account.
*   **F-FAU-02 (Must)**: As a user, I want to **Login** and have my session persist so I don't have to log in every time.
*   **F-FAU-03 (Should)**: As a user, I want to see a **loading state** during auth so I know the app is working.

---

## Acceptance Criteria
- [x] User can see Register page.
- [x] User can see Login page.
- [x] Form validation works (email format, password length).
- [x] Successful login redirects to Dashboard.
- [x] Token is stored in localStorage/cookie for persistence.
- [x] AuthGuard redirects unauthenticated users to /login.
