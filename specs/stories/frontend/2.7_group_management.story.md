# Story: Group Management

**Feature ID**: F-FGR
**Epic**: Group Management
**Status**: TODO
**Owner**: Frontend Agent

---

## User Stories
*   **F-FGR-01 (Should)**: As a user, I want to **create a group** and select multiple members from a search list.
*   **F-FGR-02 (Should)**: As a user, I want to **leave a group** so I no longer receive messages from it.
*   **F-FGR-03 (Should)**: As a group admin, I want to **delete the group** entirely.
*   **F-FGR-04 (Must)**: As a user, I want to **view group information** (member list, etc) to know who I am chatting with.
*   **F-FGR-05 (Must)**: As a group admin, I want to **add or remove members** from the group.

---

## Acceptance Criteria
- [x] Create Group modal/screen allowing name input and member selection.
- [ ] Group Info screen showing members.
- [ ] Ability for Admin to add/remove members in Info screen.
- [ ] User can leave group from Info screen.
- [ ] Admin can delete group from Info screen.
